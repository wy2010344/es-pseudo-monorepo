# Apps

这个目录包含应用项目，可能是私有的，每个子目录都是一个独立的 GitHub 仓库。

## 结构

```
apps/
├── mve-vite-demo/    # MVE Vite 演示应用
└── ...               # 其他应用
```

## 特点

- 应用可以引用 `packages/` 下的包
- 使用 `workspace:*` 引用本地包进行开发
- 支持热重载和快速开发
- 可以是私有仓库

## 开发流程

1. 在应用目录下运行 `pnpm run dev` 开始开发
2. 修改 `packages/` 下的包会自动反映到应用中
3. 使用统一的代码风格和 Git 钩子
